# Day027 — Pixel Art Editor

## 概要

TypeScript + Vite + Canvas API で作ったブラウザ動作のピクセルアートエディタ。
16 色パレット・フラッドフィル・Undo/Redo・PNG エクスポートを備えた、外部依存ゼロのクリエイティブツール。

## 技術スタック

- Language: TypeScript
- Framework/Library: Vite（バンドラーのみ）
- その他: Canvas API（外部依存なし）

## 起動方法

```bash
# セットアップ
npm install

# 開発サーバー
npm run dev

# 本番ビルド
npm run build
```

ブラウザで http://localhost:5173 を開く。

## 開発ログ

### 学んだこと

- Canvas の `image-rendering: pixelated` で拡大表示時のぼやけを防止できる
- Flood Fill はキューベースの BFS で実装するとシンプルかつ高速
- Canvas の `getImageData` で描画後の色を検証してテストできる

### 詰まったこと・解決方法

- 透明ピクセルの表現：`null` で透明を表し、レンダリング時にチェッカーパターンを描画することで視覚的に識別可能にした
- タッチイベントの `passive: false`：`preventDefault()` を呼ぶためリスナーを非パッシブにする必要があった
