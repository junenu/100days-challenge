# Day029 — Mandelbrot Explorer

## 概要

ブラウザで動作するインタラクティブなマンデルブロ集合エクスプローラー。
Canvas API を使ったスムース・カラーリングで美しいフラクタルを描画し、マウス・タッチで自由にズーム/パンできる。

## 技術スタック

- Language: TypeScript
- Framework/Library: Vite（外部描画ライブラリなし）
- Rendering: Canvas API（CPU レンダリング）
- その他: スムースエスケープカウント法、5 種カラーマップ

## 起動方法

```bash
# セットアップ
npm install

# 開発サーバー
npm run dev

# ビルド
npm run build
```

## 操作方法

| 操作 | 動作 |
|------|------|
| ホイール / ピンチ | ズームイン/アウト |
| クリック | クリック位置を中心に 2 倍ズーム |
| ドラッグ / スワイプ | パン（移動） |
| `+` / `-` | ズームイン/アウト |
| 矢印キー | パン |
| `R` | 初期ビューにリセット |

## 開発ログ

### 学んだこと

- **スムースカラーリング**: 整数のエスケープカウントではなく `log(log(|z|)) / log(2)` を使って連続的な値に変換することで、バンドが消えて滑らかなグラデーションになる
- **ズーム中心の保持**: ズーム時に「マウス位置の複素数座標が変わらない」ように中心を補正する式が重要

### 詰まったこと・解決方法

- DPR 対応で canvas の論理サイズと物理サイズが混在してバグ → `clientWidth/Height` と `canvas.width/height` を明確に分離
- ズーム時のパン方向が逆になる → `centerX = cx + (old_cx - cx) / factor` の式で修正
