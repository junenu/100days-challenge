# Day026 — life

## 概要

ターミナルで動く Conway's Game of Life シミュレーター。
ANSI エスケープコードのみで描画し、外部ライブラリ不使用。
新生セルと生存セルを色分け表示、7 種類のプリセットパターン内蔵。

## 技術スタック

- Language: Node.js (ESM)
- Framework/Library: なし（標準ライブラリのみ）
- Terminal: ANSI エスケープコード + raw mode stdin

## 起動方法

```bash
node src/index.js
```

※ 対話式ターミナル（TTY）で実行する必要があります。

## 操作キー

| キー | 動作 |
|------|------|
| `Space` | 一時停止 / 再開 |
| `s` | 1 ステップ進める（停止中のみ有効） |
| `r` | ランダム配置にリセット |
| `c` | 全セルをクリア |
| `+` / `=` | 速度アップ（間隔 -50ms） |
| `-` | 速度ダウン（間隔 +50ms） |
| `1` | Glider |
| `2` | Blinker |
| `3` | Toad |
| `4` | Beacon |
| `5` | Pulsar |
| `6` | Gosper Glider Gun |
| `7` | Pentadecathlon |
| `q` / Ctrl+C | 終了 |

## 開発ログ

### 学んだこと

- `process.stdin.setRawMode(true)` でキーイベントを文字単位で受け取れる
- `setInterval` より再帰 `setTimeout` のほうが可変インターバルに対応しやすい
- `Uint8Array` でグリッドを持つと通常の配列より省メモリ
- `CLEAR_SCREEN + HOME` で毎フレームフルクリア描画するシンプルな方法が効果的

### 詰まったこと・解決方法

- `setRawMode is not a function`: 非 TTY 環境（`timeout` コマンド等）では stdin が TTY でない。`isTTY` チェックを追加
- セル描画幅：全角 2 文字（`██`）を使うことでターミナルのフォント幅に自然にフィット
